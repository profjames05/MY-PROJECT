# Logo Setup Instructions

## How to Add the UENR Logo

1. **Save the Logo File:**
   - The UENR logo image should be named: `uener_logo.png`
   - Save it in the `images/` folder of your project
   - Full path: `c:\Users\PROF JAMES\OneDrive\Desktop\Attendance_Tracker\images\uener_logo.png`

2. **Logo Specifications:**
   - Recommended size: 200px width (will auto-scale height)
   - Format: PNG with transparency recommended
   - The logo will display at the top of the page for easy identification

3. **Verify Setup:**
   - Start the server: `node server.js`
   - Open: `http://localhost:3000`
   - If you see the logo at the top, setup is complete!
   - If you don't see the logo, ensure the file is at the correct path
